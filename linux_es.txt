VIGASOCO (VIdeo GAmes SOurce COde) Linux SDL port (v0.04)

Acerca de:
==========

VIGASOCOSDL es la versión para SDL del proyecto VIGASOCO.
Se ha probado en Linux, aunque no debe ser dificil adaptarlo a cualquier otra plataforma con SDL y un compilador moderno de C++.


Licencia y distribución:
========================

La misma que el proyecto VIGASOCO (consultar leeme.txt para los detalles).

Uso:
===

Similar a la versión Windows ( ver window_es.txt).
Las principales diferencias son:
	- No existe soporte para el pacman.
	- No hay loader para zip, la ROM tiene que estar descomprimida.
